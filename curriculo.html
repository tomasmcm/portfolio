ENG

The aim of this project is to build an interactive online curriculum. The inspiration for this website was the operating system Macintosh OS 6 (the most popular OS from 1 to 7). This OS dates from 1988 and was one of the first systems that had features that today we can't live without such as multitasking, shortcuts, desktop. I decided to make a tribute to these mythical OS using icons and graphics from the system itself.
The website is divided into 3 main sections (About, Skills & Hobbies) plus the main screen and the contacts. The section About contains information on my training and education. The section Skills presents in the form of graphs the various programs and programming languages that know. Finally, the section Hobbies lists some hobbies and personal likes.

PT

A proposta para este projecto consiste na construção de um currículo online interactivo. A fonte de inspiração para este website foi o sistema operativo Macintosh OS 6 (o mais popular dos OS 1 a 7). Este OS data de 1988 e foi um dos primeiros sistemas que que possuía funcionalidades que actualmente consideramos o default como multitasking, atalhos, desktop. Decidi fazer uma homenagem a esses míticos OS utilizando ícones e gráficos do próprio sistema.
O site está dividido em 3 secções principais (About, Skills e Hobbies) às quais acresce o ecrã principal e os contactos. A secção about contem informações sobre a minha formação e educação. A secção skills apresenta sob a forma de gráficos os vários programas e linguagens de programação que domino. Finalmente, a secção hobbies apresenta alguns passatempos e gostos pessoais.
